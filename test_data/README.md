# Test Data Directory

This directory contains test files used for development and testing purposes.

## Files

- `test_video.mp4` - Sample video file for testing video upload and processing functionality
- `valid_test_image.png` - Sample image file for testing image upload and processing functionality

## Usage

These files are used by various test scripts and development tools to verify that:
- Video upload functionality works correctly
- Image processing and thumbnail generation works
- File storage and retrieval systems function properly
- AR content creation with media files works as expected

## Note

These files are intentionally included in the repository for testing purposes. They should not be used in production environments.
