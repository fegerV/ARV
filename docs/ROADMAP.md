## CONTRIBUTING

### Как внести вклад

Мы рады любому вкладу в развитие проекта ARVlite! Вот как вы можете помочь:

1. **Форкните репозиторий** и создайте свою ветку из `main`
2. **Сделайте изменения** вашей ветке
3. **Напишите тесты** для новых функций или исправлений
4. **Убедитесь, что все тесты проходят** перед отправкой
5. **Создайте pull request** в основной репозиторий

### Процесс PR

1. **Создание PR**: Убедитесь, что ваш PR решает конкретную проблему или добавляет новую функцию
2. **Описание**: Предоставьте подробное описание изменений и их назначения
3. **Тестирование**: Убедитесь, что все тесты проходят и покрытие кода не снижается
4. **Согласование**: Ваш PR должен быть рассмотрен минимум одним другим участником команды
5. **Объединение**: После одобрения PR будет объединен в основную ветку

### Code review

Процесс код-ревью важен для обеспечения качества кода:

- Проверяйте читаемость и структуру кода
- Обращайте внимание на потенциальные проблемы безопасности
- Убедитесь, что новые функции покрыты тестами
- Следуйте принятому стилю кодирования проекта
- Предлагайте улучшения конструктивным образом
